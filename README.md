# Sborka Mods

Репозиторий модов для **Sborka Launcher**. Лаунчер автоматически скачивает моды из этого репозитория.

## Как обновить моды

1. Положи `.jar` файлы модов в папку `mods/`
2. Запусти скрипт генерации манифеста:
   ```bash
   node update-manifest.js
   ```
3. Закоммить и запушь:
   ```bash
   git add .
   git commit -m "Обновление модов"
   git push
   ```

Готово! При следующем запуске лаунчера все игроки автоматически получат обновлённые моды.
